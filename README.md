### 1️⃣ What is the difference between var, let, and const?

var, let and const used to declare variable, but they have different behavior. var is function-scoped. var can be redeclare and reassigned. var hoisted with undefined value. let and const both are block-scoped. with let value can be reassigned but not redeclared. but const can't be redeclarer or reassigned. let and const both are hoisted but in Temporal Dead Zone.

### 2️⃣ What is the spread operator (...)?

The spread operator expand or spread elements of an array or object. it is useful for copying or marging array/object without keeping reference to the original.

### 3️⃣ What is the difference between map(), filter(), and forEach()?

These all are array methods to iterate an array but have some differences. map() goes through each element and return a new modified array. filter() retruns a new array based on condition. forEach() loop over all the elements but doesn't return anything.

### 4️⃣ What is an arrow function?

An arrow function is a shorter syntax to write a function using => arrow syntax in javaScript. it doesn't have its own this, instead it inherit this from its surrounding scope.

### 5️⃣ What are template literals?

Template literal is a way to write string in JavaScript using backtick (``) syntax. it allows multi line string and write expressions inside the string using ${}.
